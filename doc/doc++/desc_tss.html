Esta union representa un descriptor (entrada) dentro de la GDT, debido a que la misma puede contener diferentes tipos de descriptores, sera necesario agregar a esta union todos dichos posibles tipos, entonces cuando se referencie una variable de tipo "descriptor" habra que tener en cuenta a que tipo de descriptor se esta referenciando, para ello hay que explicitarlo: descriptor desc; desc.desc_tss // referenciara a un descriptor de un TSS desc.desc_seg // referenciara a un descriptor de un segmento
