parsear_argumentos: Funcion que se encarga de parsear los argumentos, o sea de separar cuando encuentra un espacio (' ') los argumentos y asignarlos al array de argumantos.
Parameters:
comando - dato ingresado por teclado por el usuario
argumentos - [] direccion del array de argumentos donde se van a setear los mismos luego del parseo. Ej: Recibe: 'ls -l -a' antes: comando ls -l -a despues comando = ls argumentos[0] = -l argumentos[1] = -a argumentos[2] = NULL
