sys_matar: System call encargada de matar un proceso. Recibe un pid del proceso y lo mata.
Returns:
estado_fin pid del proceso matado por OK, -1 pid invalido, -2 proceso de sistema, imposible matar.
Parameters:
pid - Pid del proceso a matar.
