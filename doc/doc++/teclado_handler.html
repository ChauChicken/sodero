void teclado_handler( dword irq )

teclado_handler: Funcion que representa a la tarea que se ejecuta cuando ocurre una interrupcion de teclado (IRQ1)

Documentation

teclado_handler: Funcion que representa a la tarea que se ejecuta cuando ocurre una interrupcion de teclado (IRQ1). Se encarga de validar la tecla que se ingreso y manejar la escritura y desplazamiento en el buffer de teclado.
Parameters:
irq - Nro de IRQ la cual llama a la funcion.
