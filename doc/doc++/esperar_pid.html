esperar_pid: API que se utiliza para que un proceso espere que termine otro y luego recien continuar su ejecucion. Recibe el pid del proceso a esperar como parametro. Se llama a la system call sys_ejecutar_pid
Returns:
estado_fin pid del proceso que espere por OK. -1 por ERROR.
Parameters:
pid - pid del proceso a esperar.
