void setear_interrupcion( int numero, handler_interrupcion interrupt )

Funcion que setea una interrupcion para que sea ejecutada como un TASK GATE

Documentation

Funcion que setea una interrupcion para que sea ejecutada como un TASK GATE. Es complementaria a setear_interrupt ().
Parameters:
numero - numero de IRQ a setear (relativa a 0x20)
interrupt - puntero a la funcion que representa la tarea de atencion de la IRQ.
