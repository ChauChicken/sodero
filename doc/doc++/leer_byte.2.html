byte leer_byte()

Documentation

leer_byte: leee una palabra desde el registro FIFO del FDC unicamente cuando los bits RQM y DIO del MSR estan en uno. Echa en base al pseudocodigo del manual del 82077AA.
Returns:
palabra leida o 0xFF ( -1 ) en caso de error
