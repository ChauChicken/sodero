int leer_sector(dword cabeza, dword pista, dword sector, void* buff)

leer_sector: Lee un sector de disco, recibiendo como argumentos cabeza, pista y sector

Documentation

leer_sector: Lee un sector de disco, recibiendo como argumentos cabeza, pista y sector. Analogo a la int 13 del BIOS Implementada en base al pseudo codigo del manual de intel
Returns:
el estado de la lectura 
Comando leer datos
+---+---+---+---+---+---+---+---+
| 7 | 6 | 5 | 4 | 3 | 2 | 1 | 0 |
+---+---+---+---+---+---+---+---+
|   |   |           |   |
|   |   |           +---+--> si 11 entonces voy a leer
|   |   |
|   |   +--> si = 1 -> Skip Control
|   +--> si = 1 -> doble densidad
+--> si = 1 -> multi track 
palabra de control = 11100110 = 0xE6 (con multitrack)
palabra de control = 01100110 = 0x66 (sin multitrack)
Parameters:
cabeza - la cabeza a leer
pista - la pista a leer
sector - el sector a leer
buff - espacio en memoria donde se almacena el sector leido
