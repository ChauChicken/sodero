salir: API que se utiliza para que termine o finalice el proceso actual

Documentation

salir: API que se utiliza para que termine o finalice el proceso actual. Recibe como parametro el valor con el cual finaliza el proceso o que retorna luego de que concluya su ejecucion. Se llama a la System call sys_salir.
Returns:
estado_fin 0 por OK. -1 por ERROR.
Parameters:
retorno - valor de retorno con cual concluye el proceso actual.
